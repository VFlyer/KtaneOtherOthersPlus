On the Subject of Coding Effiencitly

Apparently you can learn coding while defusing a bomb


  • Coding Effiencitly will activate at some point.
  • When Coding Effiencitly activates, it will challenge you with a question.
  • You have twenty seconds to correctly answer the question.
  • If you incorrectly answer or the time runs out, a strike will be given.
  • good luck.
